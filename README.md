# BookLounge

# Создайте базу данных PostgreSQL

# Создайте .env файл и заполните его по следующему шаблону:
```
PORT=

SESSION_SECRET=

DATABASE_HOST=
DATABASE_PORT=
DATABASE_USERNAME=
DATABASE_PASSWORD=
DATABASE_NAME=

CONFIRMATION_CODE=

JWT_SECRET=
```

# Установите зависимости:
`npm install`

# Запустите проект:
`nest start`
